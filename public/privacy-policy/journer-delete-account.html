계정 및 데이터 삭제 안내

Journer - 여행자의 다이어리

최종 업데이트: 2026년 7월 21일

Journer 사용자는 앱 안에서 계정과 연결 데이터를 직접 삭제할 수 있습니다. 계정 삭제는 복구할 수 없습니다.

1. 앱에서 계정을 삭제하는 방법

  1. Journer에 로그인합니다.
  2. 프로필 → 설정 → 데이터와 계정 → 계정 탈퇴로 이동합니다.
  3. 삭제 대상과 복구 불가 안내를 확인합니다.
  4. 확인 체크박스를 선택하고 안내된 확인 문구를 입력합니다.
  5. 계정 탈퇴를 누르고 완료 안내가 표시될 때까지 앱을 닫지 않습니다.

서버 삭제가 완료되면 Supabase 인증 계정과 Journer 프로필, 연결된 서버 데이터가 삭제되고 카카오 또는 네이버 연결 및 기기의 해당 계정 캐시가 정리됩니다.

2. 계정 삭제 시 삭제되는 데이터

  • Supabase 인증 계정과 Journer 프로필
  • 사용자가 소유한 여행, 일정, 장소, 비용, 예약, 체크리스트 및 활동 내역
  • 사용자가 작성한 댓글·답글, 북마크, 멘션 및 알림 설정
  • Firebase 기기 푸시 토큰
  • 사용자가 직접 업로드한 사진과 소유 여행의 비공개 파일

3. 별도 보관되는 데이터와 기간

추가 보관 기간: 없음. 현재 Journer는 계정 삭제가 완료된 뒤 사용자 데이터를 별도로 보관하지 않습니다. 다만 법령상 보존 의무가 새로 발생하는 경우에는 해당 법정 기간 동안 필요한 최소 정보만 분리 보관한 뒤 삭제합니다. 다른 동행자가 별도로 작성한 콘텐츠는 해당 동행자의 계정과 권리에 따라 유지될 수 있습니다.

4. 계정을 삭제하지 않고 일부 데이터만 삭제하는 방법

  • 여행 상세 또는 편집 화면에서 여행 기록과 하위 일정을 삭제할 수 있습니다.
  • 본인이 작성한 댓글과 답글을 삭제할 수 있습니다.
  • 업로드한 사진을 여행 기록에서 삭제할 수 있습니다.
  • 여행별 알림과 공개 범위를 설정에서 변경할 수 있습니다.

5. 앱에 접근할 수 없는 경우

아래 이메일로 계정 또는 데이터 삭제를 요청해 주세요. 비밀번호를 보내지 말고, Journer 계정에 연결된 이메일 주소와 요청 내용을 알려주세요. 계정 소유 확인 후 지체 없이 처리하며, 일반적으로 요청 접수 후 7일 이내에 결과를 안내합니다.
