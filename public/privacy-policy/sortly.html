Sortly - 스마트 파일 정리 앱

시행일 및 최종 업데이트: 2026년 7월 21일

핵심 안내: 파일 정리와 로컬 분석은 기본적으로 사용자의 PC 안에서 처리됩니다. 외부 전송은 Microsoft Store 기능, 사용자가 직접 동의한 사용 통계, 업데이트 확인 또는 사용자가 동의한 클라우드 AI 기능에 필요한 범위에서 발생합니다.

1. 처리자 및 적용 범위

본 방침은 Microsoft Store에서 제공되는 Windows용 Sortly 앱과 Sortly 관련 웹페이지에 적용됩니다.

  • 운영자 및 개인정보 보호책임자: Choi Seong Hyeon (Microsoft Store 게시자명: choioki)
  • 문의: choidev98@gmail.com
  • 계정: Sortly는 별도의 회원가입이나 로그인을 요구하지 않습니다.

다음 정보는 앱 기능 제공을 위해 사용자의 기기에서 처리·저장되며, 아래에서 별도로 설명한 경우를 제외하고 운영자 서버로 전송되지 않습니다.

3. 사용 통계

제품 개선을 위한 사용 통계는 기본적으로 꺼져 있습니다. Microsoft Store 설치본에서 사용자가 앱 설정 → 사용 통계 전송(선택)을 직접 켠 경우에만 전송되며, 언제든 다시 끌 수 있습니다. 이 설정을 끄면 이후 통계 이벤트 전송은 중단됩니다. 이미 수집된 기록의 삭제는 아래 문의처로 요청할 수 있습니다.

  • 수집 항목: 무작위 설치 ID와 세션 ID, 이벤트명, 앱 버전, 운영체제, 언어, 플랜, Store 설치 여부, 기능 실행 결과, 오류 유형, 개수 구간, 업데이트·구매 버튼 상호작용
  • 수집하지 않는 항목: 파일명, 폴더 경로, 파일·문서 내용, 이름, 이메일, 결제수단 정보
  • 목적: 활성 설치 수와 기능 이용 추세 파악, 오류 분석, 제품 및 구매 흐름 개선
  • 보관: 이벤트 발생일부터 최대 24개월을 원칙으로 하며, 그 이후에는 삭제하거나 개인 또는 기기를 구분할 수 없는 집계 형태로 전환합니다.

무작위 설치 ID는 이름이나 이메일을 포함하지 않지만, 동일 설치본의 이벤트를 연결하는 지속 식별자이므로 본 방침에서는 단순히 “완전한 익명 정보”라고 표현하지 않습니다.

4. Microsoft Store 구독 상태 동기화

유료 기능 제공, 라이선스 상태 확인 및 구독 지원을 위해 활성 Store 구독 정보가 운영자의 Supabase 기반 관리 저장소로 전송될 수 있습니다. 이 처리는 사용 통계 설정과 별개이며, 활성 유료 구독자에게만 적용됩니다.

  • 항목: 앱 전용 무작위 게시자 사용자 ID, 앱 버전, Microsoft가 검증한 Store 구독·상품·SKU 식별자, 구매 국가/지역, 활성·갱신·평가판 상태, 시작·만료·취소·최종 변경 시각
  • 수집하지 않는 항목: Microsoft 계정 이메일, 사용자 이름, 카드 번호, 청구지 주소
  • 보관: 구독 제공 및 분쟁 대응을 위해 최종 갱신·변경일을 기준으로 최대 5년 보관한 뒤 자동 삭제합니다. 법령상 별도 보존 의무가 적용되는 기록은 해당 기간 동안 분리 보관할 수 있습니다.

5. 선택형 클라우드 AI

클라우드 AI는 기본적으로 꺼져 있으며, 사용자가 안내를 확인하고 직접 시작한 경우에만 다음 정보가 HTTPS로 전송됩니다.

  • AI 파일 분류: 파일명, 사용자가 설정한 분류 폴더 목록, 지원 문서에서 추출한 앞부분 텍스트 일부
  • 문장으로 규칙 만들기: 사용자가 입력한 지시문과 대상 파일의 이름, 확장자, 크기, 경과일
  • 이미지·음악: 이미지 원본과 음악 본문은 AI 문서 분석 목적으로 전송하지 않으며 로컬 메타데이터를 사용합니다.

요청은 Sortly의 Cloudflare Worker를 거쳐 OpenAI API로 전달됩니다. Sortly 운영자는 요청 본문을 자체 데이터베이스에 저장하도록 구현하지 않았습니다. OpenAI API 데이터는 기본적으로 모델 학습에 사용되지 않으며, 서비스 안전을 위한 남용 모니터링 로그에 고객 콘텐츠가 포함되어 최대 30일간 보관될 수 있습니다. 공급자의 법적 의무나 계정별 데이터 제어 설정에 따라 예외가 있을 수 있습니다.

사용자는 클라우드 AI 제공을 거부하거나 사용을 중단할 수 있습니다. 이 경우 로컬 정리 기능은 계속 사용할 수 있지만 클라우드 AI 분류 및 문장 기반 규칙 기능은 사용할 수 없거나 로컬 방식으로 제한될 수 있습니다.

6. 외부 서비스, 처리위탁 및 국외 처리

수신자/처리자처리 내용과 목적처리 위치·기간
MicrosoftStore 설치·업데이트·결제·구독 및 라이선스 처리Microsoft가 고지한 국가와 기간. Microsoft가 독립적으로 처리하는 정보에는 Microsoft 개인정보 처리방침이 적용됩니다.
Supabase, Inc.사용 통계와 활성 Store 구독 상태의 전송·저장·관리미국 및 선택된 프로젝트 리전·하위처리자 소재국. 통계는 최대 24개월 원칙, 구독 기록은 제4항의 기간
Cloudflare, Inc.선택형 AI 요청 전달과 네트워크 보안·전송 제공미국 및 글로벌 네트워크 처리 위치. AI 요청 본문은 운영자 데이터베이스에 저장하지 않으며, 네트워크·보안 로그는 Cloudflare 정책에 따름
OpenAI, L.L.C.선택형 AI 분류와 규칙 해석미국 등 OpenAI 처리 위치. 기본 남용 모니터링 로그는 최대 30일
GitHub, Inc.Store 설치본의 업데이트 버전 확인 파일, 비-Store 설치본의 업데이트 확인·다운로드, 본 방침 등 공개 웹페이지 제공미국 및 GitHub 인프라 위치. 접속 로그는 GitHub 정책에 따름

위 서비스 이용 과정에서 IP 주소, 요청 시각, 사용자 에이전트 등 일반적인 네트워크 로그가 각 공급자의 보안·운영 목적으로 자동 처리될 수 있습니다. Sortly는 개인정보를 판매하거나 맞춤형 광고에 이용하지 않습니다.

전송 시점·방법: 사용 통계는 사용자가 동의한 뒤 해당 이벤트가 발생할 때, 구독 정보는 활성 Store 라이선스 확인 시, AI 정보는 사용자가 클라우드 AI 작업을 요청할 때, 업데이트 정보는 앱 실행·업데이트 확인·웹페이지 접속 시 HTTPS 네트워크 요청으로 전송됩니다.

거부 방법과 영향: 사용 통계는 처음부터 동의하지 않거나 앱 설정에서 끌 수 있고, 클라우드 AI는 동의하지 않거나 사용하지 않을 수 있습니다. 이 경우 해당 통계 전송 또는 AI 기능만 중단됩니다. 활성 구독 동기화를 차단하면 구독 지원이 제한될 수 있고, 업데이트 서버 연결을 차단하면 업데이트 확인·다운로드가 정상 동작하지 않을 수 있습니다. Microsoft가 Store 거래 과정에서 직접 처리하는 정보는 Microsoft 계정과 Store 설정을 따릅니다.

7. 삭제와 권리 행사

  • 앱에서 규칙, 필터, 제외 폴더, 이동 기록 및 관련 설정을 삭제하거나 변경할 수 있습니다.
  • 앱을 제거한 뒤에도 운영체제 설정에 따라 일부 로컬 앱 데이터가 남을 수 있으며, 필요한 경우 Windows 앱 데이터 관리 기능을 통해 제거할 수 있습니다.
  • 서버에 저장된 사용 통계 또는 구독 동기화 기록의 열람·정정·삭제·처리정지를 요청하려면 아래 이메일로 연락해 주세요. 요청자를 확인하기 위해 설치 ID나 구독 식별 정보의 일부를 요청할 수 있습니다.
  • 법령상 보관 의무가 있는 정보는 해당 기간 동안 분리 보관한 뒤 삭제할 수 있습니다.

8. 안전성 확보 조치

외부 통신에는 HTTPS를 사용하며, 서버 저장소에는 접근권한 제한, 관리자 전용 조회, 데이터베이스 행 수준 보안 및 비밀키 분리 등의 조치를 적용합니다. 다만 인터넷 전송이나 전자 저장 방식의 절대적인 안전을 보장할 수는 없습니다.

9. 아동의 개인정보

Sortly는 만 14세 미만 아동을 대상으로 설계되지 않았으며, 법정대리인의 동의 없이 아동의 개인정보를 의도적으로 수집하지 않습니다. 관련 사실을 알게 된 경우 문의처로 연락해 주세요.

10. 방침 변경

처리 항목, 목적, 외부 서비스 또는 보관 기준이 중요하게 변경되는 경우 이 페이지의 업데이트 날짜를 변경하고, 필요한 경우 앱이나 Microsoft Store 설명을 통해 추가로 알립니다.
