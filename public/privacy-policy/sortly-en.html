Privacy Policy

Sortly - Smart File Organizer

Last Updated: July 10, 2026

Sortly does not collect file names, folder paths, document contents, or email addresses for usage analytics. File organization rules and move history are stored locally on your device by default.

1. Information Stored or Collected

Sortly does not require sign-up or login. The following data may be stored locally on your device to provide app features.

  • File organization rules: file types, name keywords, filters, and excluded folder settings configured by the user
  • Folder watch settings: watched folders and scheduled cleanup settings selected by the user
  • File move history: move history required for undo functionality
  • App settings: language, theme, subscription state, AI usage, and other preferences

2. AI Organize Processing

When you use AI organize, file names and snippets from supported documents may be sent through Sortly's AI processing server to OpenAI to generate sorting suggestions.

  • The purpose of this transfer is to generate file classification results.
  • Sortly does not separately store file names or document contents sent for AI organize.
  • Images and music files are processed on-device and their contents are not sent for AI document analysis.
  • AI organize is used only when you directly run or select the related feature.

3. Anonymous Usage Analytics

Sortly may collect anonymous usage analytics to improve the product. You can turn off anonymous usage analytics in the app settings.

The following information may be collected:

  • Whether the app was opened and the date of use
  • App version, operating system, language, and subscription plan
  • Feature usage type, such as organize, preview, AI organize, duplicate scan, undo, and Pro tools
  • Non-identifying summary information such as success/failure, error type, and file count ranges
  • A random anonymous install ID and session ID used to avoid duplicate counting

Anonymous usage analytics does not collect:

  • File names
  • Folder paths
  • File or document contents
  • Email addresses, names, or contact information
  • Information that directly identifies an individual user

4. How We Use Information

  • Providing automatic file organization, undo, folder watching, and scheduled cleanup features
  • Generating AI organize results and managing AI usage limits
  • Improving app stability and understanding commonly used features
  • Analyzing errors and improving the product
  • Maintaining user settings

5. Third-Party Services

Sortly may use the following services, and each service's own privacy policy may also apply.

  • Microsoft Store: app installation, updates, subscription payments, and license verification
  • OpenAI: analysis of file names and supported document snippets for AI organize
  • Cloudflare Worker: operating AI request and anonymous usage analytics endpoints
  • Nopl self-managed analytics store: aggregating anonymous usage analytics and providing an admin dashboard

6. Retention and Deletion

  • Local data is stored on your device and can be deleted by uninstalling the app or resetting settings.
  • Anonymous usage analytics may be retained for the period necessary to improve the product.
  • File names and document contents sent for AI organize are not separately stored by Sortly.
  • You may contact us using the email below for privacy or analytics deletion requests.

7. Your Rights and Choices

  • You can turn off anonymous usage analytics in the app settings.
  • You can delete saved rules, excluded folders, and move history in the app.
  • You can remove local data by uninstalling the app.
  • For privacy-related inquiries, please contact us at the address below.

8. Children's Privacy

Sortly is not directed to children under 13 and does not knowingly collect children's personal information.

9. Changes to This Policy

If this policy is updated, we will indicate this through the "Last Updated" date on this page. For material changes, we may provide additional notice in the app or Store listing.
