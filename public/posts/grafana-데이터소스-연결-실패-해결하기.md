# Grafana 데이터소스 연결 실패 해결하기

---

## 증상 및 오류 메시지
Grafana에서 데이터소스를 설정할 때 연결 실패 오류가 발생하는 경우가 있습니다. 일반적으로 아래와 같은 오류 메시지를 확인할 수 있습니다.

```text
Data source is not available
```
또는
```text
Error: Cannot connect to the database
```
이러한 오류는 데이터베이스와의 연결 문제로 인해 발생하며, Grafana 대시보드에서 데이터를 시각화할 수 없게 됩니다.

## 대표 원인
이 문제의 원인은 여러 가지가 있을 수 있습니다. 가장 일반적인 원인은 다음과 같습니다.
1. **잘못된 데이터베이스 URL**: Grafana 설정에서 데이터베이스 URL이 올바르지 않을 수 있습니다.
2. **인증 정보 오류**: 사용자 이름이나 비밀번호가 잘못 입력되었을 수 있습니다.
3. **방화벽 설정**: 데이터베이스 서버의 방화벽이 Grafana 서버의 IP를 차단하고 있을 수 있습니다.
4. **데이터베이스 서버 다운**: 데이터베이스 서버가 작동하지 않거나 네트워크 문제로 인해 접근할 수 없을 수 있습니다.

## 문제 확인 명령어
문제를 해결하기 위해 다음과 같은 명령어로 데이터베이스 연결 상태를 확인할 수 있습니다. 예를 들어 MySQL 데이터베이스의 경우:

```bash
mysql -u 사용자명 -p -h 데이터베이스_IP -P 포트번호
```

이 명령어로 데이터베이스에 직접 연결해 보세요. 연결이 성공하면 Grafana 설정의 문제일 가능성이 높습니다. 연결이 실패한다면 데이터베이스 서버의 상태를 점검해야 합니다.

## 해결 절차
1. **데이터베이스 URL 확인**: Grafana의 데이터소스 설정에서 데이터베이스 URL이 올바른지 확인합니다. 예를 들어 MySQL의 경우는 다음과 같이 설정해야 합니다.
   ```text
   jdbc:mysql://데이터베이스_IP:포트/데이터베이스명
   ```

2. **인증 정보 확인**: 사용자 이름과 비밀번호가 정확한지 다시 확인합니다. Grafana 설정에서 입력된 정보가 데이터베이스와 일치하는지 확인합니다.

3. **방화벽 설정 확인**: 데이터베이스 서버의 방화벽 설정을 점검하여 Grafana 서버의 IP가 허용되어 있는지 확인합니다. 필요한 경우 방화벽 규칙을 수정합니다.

4. **데이터베이스 서버 상태 확인**: 데이터베이스 서버가 정상적으로 작동 중인지 확인합니다. 서버가 다운되었거나 네트워크 문제로 인해 접근할 수 없는 경우, 서버를 재시작하거나 네트워크 문제를 해결해야 합니다.

5. **로그 확인**: Grafana와 데이터베이스의 로그를 확인하여 추가적인 오류 메시지를 찾아보세요. Grafana의 로그는 일반적으로 `/var/log/grafana/grafana.log`에 위치합니다. 이 로그에서 데이터소스 연결 관련 오류를 확인할 수 있습니다.

## 흔한 실수
- **URL 포맷 오류**: 데이터베이스 URL을 입력할 때 오타나 형식 오류가 발생할 수 있습니다. 특히, 포트 번호나 IP 주소를 잘못 입력하는 경우가 많습니다.
- **인증 정보 캐싱**: Grafana에서 이전 인증 정보를 캐시하고 있을 수 있습니다. 이 경우, Grafana를 재시작하거나 데이터소스를 삭제 후 다시 추가해 보세요.
- **네트워크 문제 간과**: 데이터베이스 서버와 Grafana 서버 간의 네트워크 연결 문제를 간과하는 경우가 많습니다. 네트워크 상태를 확인해야 합니다.

## 재발 방지 체크리스트
- 데이터소스 설정을 변경한 후 항상 연결 테스트를 수행합니다.
- 데이터베이스와 Grafana 서버 간의 네트워크 상태를 정기적으로 점검합니다.
- 방화벽 규칙을 문서화하고, 필요할 때 쉽게 수정할 수 있도록 합니다.
- Grafana와 데이터베이스의 로그를 정기적으로 검토하여 문제를 사전에 예방합니다.

이와 같은 절차를 통해 Grafana 데이터소스 연결 실패 문제를 효과적으로 해결할 수 있습니다. 문제가 지속된다면, 관련 커뮤니티나 공식 문서를 참고하여 추가적인 도움을 받는 것도 좋습니다.
