# ✏️ Vim 편집기에서 파일 전체 삭제하는 방법 정리

---

리눅스에서 작업하다 보면 vim으로 파일을 열었는데

내용을 통째로 지우고 새로 작성하고 싶을 때가 있습니다.

하지만 처음 vim을 접하면

"전체 선택이 어디 있지?"

"Ctrl+A도 안 먹는데?"

이렇게 당황하는 경우도 많죠.

오늘은 **vim에서 파일 전체 내용을 지우는 3가지 방법** 을

가장 간단하게 정리해보았습니다.

---

**✔️ 1. 가장 쉬운 방법: :%d 명령**

vim에서 전체 삭제할 때 **가장 많이 쓰는 방식** 입니다.

**👉 사용법**

  1. **ESC** 키 누르기 (명령 모드 이동)

  2. 아래 입력:

```
:%d
``` 

이 한 줄이면 **파일 전체 내용이 싹 지워집니다.**

---

**✔️ 2. 처음부터 끝까지 삭제 (gg + dG)**

키 조합을 사용해서 전체 삭제하는 방식입니다.

**👉 사용법**

  1. ESC

  2. 아래 순서대로 입력:

```
gg
``` 

→ 파일 맨 처음으로 이동

```
dG
``` 

→ 현재 위치부터 파일 끝(G)까지 삭제

이렇게 하면 전체 삭제와 동일합니다.

---

**✔️ 3. 전체 삭제 후 저장하고 종료**

파일 비운 뒤 저장하려면:

```
:wq
``` 

강제로 저장하고 나가고 싶으면:

```
:wq!
``` 

이 조합은 실무에서 정말 자주 씁니다.

---

**⭐ 핵심 요약**

---

**💡 작은 TIP**

  * vim은 **입력 모드에서는 명령이 먹지 않기 때문에**

  * 항상 먼저 ESC 를 누른 후 명령을 입력해야 합니다.

  * 전체 삭제 후 바로 새로운 내용을 입력하고 싶다면

  * :%d 후 즉시 i 버튼을 눌러 입력 모드로 들어가면 됩니다.
