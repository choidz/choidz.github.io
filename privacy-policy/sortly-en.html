Privacy Policy

Sortly - Smart File Organizer

Last Updated: January 11, 2026

Sortly values your privacy. All data is stored only on your device. No information is transmitted to external servers.

1. Information We Collect

Sortly does not collect any personal information.

All app data is stored in a local SQLite database on your computer:

  • File Classification Rules: User-defined file organization rules
  • Folder Watch Settings: List of monitored folders
  • File Move History: Movement history for undo functionality
  • App Settings: User preferences such as theme and language

2. How We Use Information

Stored data is used solely for the following purposes:

  • Providing automatic file classification service
  • Enabling file move undo functionality
  • Maintaining user settings

3. Data Storage and Security

  • All data is stored locally on your device only.
  • No data is transmitted to external servers.
  • You can completely delete all data by uninstalling the app.
  • Please keep your operating system and security software up to date.

4. Third-Party Services

Sortly does not use any third-party analytics, advertising, or tracking services.

The app works completely offline, and all features are available without an internet connection.

5. Your Rights

  • You can completely delete all data by uninstalling the app.
  • You can individually delete saved rules and history within the app settings.
  • For privacy-related inquiries, please contact us at the address below.

6. Children's Privacy

Sortly does not collect personal information from anyone, including children under 13 years of age.

7. Changes to This Policy

If this policy is updated, we will notify you through the "Last Updated" date on this page.
