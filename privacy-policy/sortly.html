개인정보 처리방침

Sortly - 스마트 파일 정리 앱

최종 업데이트: 2026년 1월 11일

Sortly는 사용자의 개인정보를 소중히 여기며, 모든 데이터는 사용자의 기기에만 저장됩니다. 외부 서버로 전송되는 정보는 없습니다.

1. 수집하는 정보

Sortly는 어떠한 개인정보도 수집하지 않습니다.

앱의 모든 데이터는 사용자 컴퓨터의 로컬 SQLite 데이터베이스에 저장됩니다:

  • 파일 분류 규칙: 사용자가 설정한 파일 정리 규칙
  • 폴더 감시 설정: 감시 대상 폴더 목록
  • 파일 이동 기록: 되돌리기 기능을 위한 이동 히스토리
  • 앱 설정: 테마, 언어 등 사용자 환경설정

2. 정보 이용 목적

저장된 데이터는 오직 다음 목적으로만 사용됩니다:

  • 파일 자동 분류 서비스 제공
  • 파일 이동 되돌리기 기능 제공
  • 사용자 설정 유지

3. 정보 보관 및 보호

  • 모든 데이터는 사용자 기기에만 로컬로 저장됩니다.
  • 외부 서버로 전송되는 데이터는 없습니다.
  • 사용자는 앱 삭제를 통해 모든 데이터를 완전히 삭제할 수 있습니다.
  • 운영 체제와 보안 소프트웨어를 최신 상태로 유지하시기 바랍니다.

4. 제3자 서비스

Sortly는 제3자 분석, 광고 또는 추적 서비스를 사용하지 않습니다.

앱은 완전히 오프라인으로 작동하며, 인터넷 연결 없이도 모든 기능을 사용할 수 있습니다.

5. 사용자 권리

  • 앱 삭제를 통해 모든 데이터를 완전히 삭제할 수 있습니다.
  • 앱 내 설정에서 저장된 규칙과 기록을 개별적으로 삭제할 수 있습니다.
  • 개인정보 관련 문의는 아래 연락처로 보내주세요.

6. 아동 개인정보

Sortly는 만 13세 미만 아동을 포함하여 누구로부터도 개인정보를 수집하지 않습니다.
