# Sortly: 스마트 파일 정리 앱 완벽 가이드

---

다운로드 폴더, 지금 어떤 상태인가요? 저는 솔직히 말하면... 파일이 수백 개 쌓여있었어요. 문서, 이미지, 압축파일, 설치파일이 뒤섞여 있고, 같은 파일을 여러 번 받아서 중복도 엄청 많고요.

"나중에 정리해야지" 하면서 미루다 보니 점점 손대기 힘든 상태가 됐죠. 이런 문제를 해결하고 싶어서 만든 게 바로 **Sortly**예요.

> **한줄 요약**: Sortly는 파일을 확장자별로 자동 분류하고, 중복 파일을 찾아주고, 실수로 이동한 파일을 되돌릴 수 있는 무료 데스크톱 앱이에요. Microsoft Store에서 다운로드

![](/images/posts/sortly-00-main.png)

_Sortly 메인 화면 - 감시 폴더와 최근 이동 기록을 한눈에 볼 수 있어요_

---

**분류 규칙 설정 - 핵심 기능**

Sortly의 핵심은 확장자 기반 자동 분류예요. 기본적으로 문서, 이미지, 비디오, 오디오, 압축파일, 설치파일 등의 카테고리가 준비되어 있어요.

![](/images/posts/sortly-01-classification.png)

_분류 규칙 화면 - 확장자별로 어느 폴더로 이동할지 설정할 수 있어요_

특히 좋은 점은:

  * 기본 규칙을 그대로 쓰거나, 내 스타일에 맞게 수정 가능

  * 새로운 카테고리 추가도 자유롭게

  * 정규식 패턴으로 파일명 기반 분류도 가능해요

  * 필요 없는 기본 규칙은 숨길 수 있어요

정리하자면, 내 작업 스타일에 맞게 분류 체계를 커스터마이징할 수 있어요.

---

**중복 파일 찾기**

다운로드 폴더에 같은 파일이 여러 개 있는 경우 많죠? Sortly는 MD5 해시 기반으로 **진짜 중복 파일**을 찾아줘요. 이름만 같은 게 아니라 내용이 완전히 똑같은 파일만 중복으로 표시해요.

![](/images/posts/sortly-02-duplication.png)

_중복 파일 검색 화면 - 같은 내용의 파일을 그룹별로 보여줘요_

기능 특징:

  * 파일 크기로 1차 필터링 후 해시 비교 (속도 최적화)

  * 중복 그룹별로 파일 목록 표시

  * 선택해서 휴지통으로 삭제 가능

  * 삭제 전 미리보기 지원

정리하자면, 불필요한 중복 파일로 낭비되는 저장 공간을 정리할 수 있어요.

---

**되돌리기 (Undo) 기능**

파일 정리하다가 "어, 이거 이동하면 안 됐는데?" 싶을 때 있잖아요. Sortly는 모든 파일 이동을 기록해두고, 언제든지 **원래 위치로 되돌릴 수 있어요**.

![](/images/posts/sortly-03-undo.png)

_이동 기록 화면 - 언제 어디서 어디로 이동했는지 기록되고, 되돌리기 가능해요_

  * 최근 이동 기록을 시간순으로 표시

  * 개별 파일 단위로 되돌리기 가능

  * 마지막 이동 일괄 되돌리기도 지원

정리하자면, 실수해도 걱정 없이 정리할 수 있어요.

---

**제외 폴더 설정**

특정 폴더는 정리 대상에서 빼고 싶을 때가 있어요. 예를 들어 진행 중인 프로젝트 폴더나, 이미 정리된 백업 폴더 같은 경우요.

![](/images/posts/sortly-04-exception.png)

_제외 폴더 설정 - 다크 모드에서 제외 폴더를 관리할 수 있어요_

  * 특정 폴더를 정리 대상에서 제외

  * 하위 폴더 포함 여부 선택 가능

  * 언제든지 제외 해제 가능

---

**폴더 감시 & 자동 정리**

Sortly는 지정한 폴더를 실시간으로 감시해요. 새 파일이 들어오면 자동으로 분류 규칙에 따라 정리해줘요.

![](/images/posts/sortly-05-organize.png)

_정리 실행 화면 - 기존 파일 일괄 정리도 가능해요_

  * 여러 폴더 동시 감시 가능

  * 시스템 트레이에서 백그라운드 실행

  * 파일 이동 시 알림 표시

  * 기존에 쌓인 파일도 일괄 정리 가능

---

**시스템 트레이 & 자동 시작**

Sortly는 시스템 트레이에 상주해요. 창을 닫아도 트레이에서 계속 파일을 감시하고 있어요.

  * 트레이 아이콘 더블클릭으로 앱 열기

  * 우클릭 메뉴에서 감시 시작/중지

  * Windows 시작 시 자동 실행 옵션

---

**다크/라이트 테마**

눈이 편한 다크 모드와 밝은 라이트 모드 중에서 선택할 수 있어요.

  * 다크/라이트 테마 전환

  * 한국어/영어 지원

  * 설정은 로컬에 저장 (개인정보 수집 없음)

---

**실무 팁**

  * **폴더 여러 개 등록하기**: 다운로드 폴더뿐만 아니라 데스크톱, 문서 폴더도 등록하면 좋아요

  * **패턴 규칙 활용하기**: "screenshot"이 포함된 파일은 스크린샷 폴더로, 이런 식의 규칙도 가능해요

  * **중복 파일 정기 검사**: 한 달에 한 번 정도 중복 파일 검사하면 저장 공간 관리에 도움돼요

  * **제외 폴더 잘 활용하기**: 작업 중인 프로젝트 폴더는 제외해두세요
