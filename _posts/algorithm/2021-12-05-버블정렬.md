제시된 <그림>은 배열 A(10)에 기억된 10개의 수치 데이터에 대하여 <br/>
버블 정렬(Bubble Sort)을 이용하여 오름차순으로 정렬 하는 순서도이다.<br/>
제시된 <처리조건>을 참조하여 <그림>의 괄 호 안 내용에 가장 적합한 항목을 <답항보기>에서 선택하시오

<br/> 

## 처리조건
    
<br/>
1. 배열 A(5,5)에 기억되는 내용은 다음과 같다.<br/>
   - N : 정렬하고자 하는 수치 데이터의 갯수    
   - i : 정렬의 회전수를 계산하기 위한 변수
   - J : 배열의 첨자 등을 위한 변수
   - FLAG : 임의의 회전 작업시 데이터의 교환이 발생하지 않을 경우 비교가 반복 되는 것을 방지하기 위한 변수
   - TM : 주 변수간의 값을 서로 바꾸기 위한 변수

2. 본 문제에서는 FLAG를 설정하여 임의의 회전 작업시 교환이 발생하지 않는다면,<br/>
정렬이 완료된 상태로 간주하고 작업을 종료시키기로 한다.

3.배열의 크기가 10일 경우 배열의 요소는 1부터 10까지 구성되는 것으로 한다. <br/>
예를들어, A라는 배열의 크기가 10일 경우 A(10)으로 표시되고,<br/>
배열 요소는 A(1) 부터 A(10)으로 구현된다고 가정한다.<br/>


<br/><br/>

## 순서도


<img src="/images/algorithm/버블정렬 순서도.png" style="width:730px; height:1456px;"><br/>
가장 첫 문제인 FLAG의 초기화 값에 대해선 어떤 값으로 초기화 해야하는지에 대해<br/>
바로 알 수 없으므로 순서도에 따라 전체적으로 파악합니다.<br/>

❖ (첫째:둘째),(둘째:셋째)...(N-1번째 : N번째) 를 비교하기 위한 <br/>
루프 조건으로 J를 하나씩 증가시키기 위해<br/>
**2 : J=J+1**<br/>
❖  (3)과 A(J+1)의 비교를 통해 (3)이 더 크다면 밑의 변수를 교환하는 것을 알 수 있습니다.<br/>
버블정렬의 정의에 따라 (3)번에는 A(J+1)의 한단계 앞의 변수인 <br/>
**3 : A(J)**<br/>
❖  밑의 변수교환에서 <br/>
**4 : A(J)=A(J+1)** <br/>

❖  이후 변경시 FLAG = 1 이 되었고, 알고리즘 처리 조건에서 데이터의 변화에 FLAG 값이 <br/>
변하지 않는다면 종료가 되도록 하였기 때문에 <br/>
**5 : FLAG**<br/>

❖  또한 이를 통해 FLAG의 초기 값 역시 <br/>
**1 : 0**<br/>

<br/>


## 코드

 ~~~
 #include <stdio.h> 
#define N 10  /* 데이터의 크기 */
void main() {int A[N] = {10, 45, 2, 6, 25, 12, 34, 12, 9, 5}; 
int i, J, FLAG, TM; 
i=-1; 
do { 
FLAG=0; 
i=i+1; 
J=-1;do { 
J=J+1;if (A[J]>A[J+1]) { /* 인접한 데이터를 비교 */
TM=A[J]; 
A[J]=A[J+1]; 
A[J+1]=TM; 
FLAG=1; /* 데이터 위치가 교환되었음을 지정 */
}} 
while (J<N-i); 

} while (i<N-1 && FLAG!=0); 
/* 정렬된 결과 출력하기 */
 for (i=0; i<N; i++) 
printf("%d ", A[i]); 
}
~~~
