## 처리조건
    
<br/>
1. 8비트 2진수 값의 최상위 비트는 부호 비트이며, 최상위 비트의 값이 0인 경우 양<br/>
수이고 1인 경우 음수이다(단, 음수의 경우 2의 보수로 표현된 값이다).<br/><br/>
2. 8비트 2진수의 값은 크기가 8인 배열 T에 T(1)부터 T(8)까지 순차적으로 저장되<br/>
어 있다.(즉, T(1)에 부호 비트가 저장되어 있음).<br/><br/>
3. 배열T의 값의 범위는 10진수 -127 ~ 127에 해당하는 2진수 값이라고 가정.<br/><br/>
4. “ABS()”는 절대값을 구하는 함수. 예를 들면 ABS(-5)의 값은 5가 된다.<br/><br/>
5. 함수“POW(a, b)”는 누승값(a의 b승)을 구하는 함수로 정의되어 있다. 예를 들 면 POW<br/>(2, 3)의 값은 2의 3승이 된다. 즉 값 8이 된다(a는 zero가 아니다.).<br/><br/>
6. “&”는 마름모내의 조건을 모두 만족하는 경우 YES, 만족하지 않을 경우는 NO로 정의한다.<br/><br/>
7. 알고리즘에 사용된 변수들은 다음과 같다.<br/>
   
- T(8) : 2진수 데이터 배열
- S : 진법변환 십진수 저장 변수 
- Sign : 부호 지정 변수
- H(8) : 보수계산을 위한 데이터 배열 
- K : 각 자리의 자리값
- B : 자리내림수 
- i : 인덱스 변수

<br/>

### ❖최상위 비트(부호 비트)가 1이면 음수
◼ 음수는 2의 보수로 표현<br/>
◼ 정확한 값을 알아보기 위해 2의 보수를 다시 취해 <br/>
양의 값을 구한 후 10진수로 변환하고 음의 부호를 붙여주면 된다.<br/><br/>
### ❖2진수의 값을 10진수로 변환하는 방법
◼ 각 자리의 수에 자리값(28-위치)을 곱해 모두 더한다.


<br/>

### ❖ 2의 보수 구하기 (1의 보수 + 1)
◼ 1의 보수를 구한 후 1을 더해서 구하게 된다.<br/>
◼ 음수의 경우 보수를 취해 양의값을 만든 후 1을 더하는 것은 <br/>
반대로 1을 뺀 후 보수를 취 하는 것과 동일한 결과를 얻을 수 있다.<br/>
◼ T(i)=0 & B=1<br/>
◼ 참(0-1의경우):앞단에서 자리 내림 발생<br/>

◼ T(1)이 1이면 -10 1110110 자리내림으로 <br/>
**1 : H(i)= T(i)-B**<br/>
◼ 앞단에서도 1을 빼주도록 B에 1을 대입<br/>
◼ 거짓(1-1의경우):더 뺄 필요가 없음<br/>
◼ B에 0을 대입<br/>
◼ 뺄셈 결과가 음수일 수 있어 절대값을 취한다 <br/>
**2 : H(i)=ABS(H(i))**<br/>
◼ 각 자리마다 1에서 빼줘야되므로<br/>
**3 : T(i)=1-H(i)**<br/><br/>
### ❖ 10진수로 변환하기
◼ 2진수의 각 자리에 자리값을 곱한 결과를 차례로 누적<br/>
**4 : K = T(i)*POW(2,8-i)** <br/>

### ❖ 원래부호계산
◼ Sign값을 곱해 원래 부호를 붙여 완성된 결과를 출력 
<br/>**5 : S=S*Sign**


## 코드

~~~
#include <stdio.h> 
#include <stdlib.h> 
#include <math.h>

void main(){
int T[8], H[8]; int S=0, K=0; int Sign, B, i;

/* 2진수 데이터를 비트 단위로 읽는다 */ 
for (i=0; i<8; i++)
scanf("%d", &T[i]);

if (T[0]!=0) { /* 음수라면 */
  Sign=-1;
  B=1; /* 뺄 값 */

/* 원래 데이터에서 B(1)을 뺀다 */ 
for (i=7; i>=0; i--) {

  H[i]=T[i]-B;
  if (T[i]==0 && B==1) /* 0-1인 경우 */
    B=1; /* 위단에서 빌려오게 된다 */ 
  else
    B=0; /* 위단에서 빌려오지 않는다 */
    H[i]=abs(H[i]); /* 절대값 */ 
  }

/* 1의 보수 : 각 자리마다 1에서 빼준다 */ 
  for (i=0; i<8; i++)
    T[i]=1-H[i]; 
  }else 
  Sign=1;

/* 10진수로 변환 : 자리값을 곱해 누적 */ 
for (i=7; i>=0; i--) {
  K=T[i]*pow(2, 7-i); 
  S=S+K;
}
S=S*Sign; /* 원래 데이터의 부호 고려 */
printf("%d\n", S);

}
