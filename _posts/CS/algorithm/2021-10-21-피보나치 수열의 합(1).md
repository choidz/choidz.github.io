제 1항부터 제 N항까지 피보나치 수열의 합(S =1 + 1 + 2
<br/> + 3 + 5 + 8 + 13 + 21 +34 + 55 +...)을 
<br/> 계산하는 알고리즘이다. 제시된 <처리조건>을 참조하여
<br/> <그림>의 괄호안에 가장 적합한 항목을 
<br/> <답항보기>에서 선택하시오.

<br/>

## 처리조건
    
<br/>
1. 계산하려는 항의 수 N을 입력하여 처리하기로 한다.(단,
N≥=10)
2. 알고리즘에 사용된 변수 등은 다음과 같다.
- A,B,C : 두개의 항 합산시 첫번째 항, 두번째 항, 기억변수
- Y : 피보나치 수열의 합을 저장하는 변수
- N : 계산하려는 항의 수 변수
- K : 인덱스 변수

<br/>
제 1항부터 제 N항까지
피보나치 수열의 합
(Y =1 + 1 + 2 + 3 + 5 + 8 + 13 + 21 +34 +...)
<br/>


◼ 초기값 A=1,B=1이 주어졌으니 <br/>
참고: 피보나치 수열 f1 = 1<br/>
f2 = 1<br/>
fn = fn-2 + fn-1 (n≥3)<br/>
**1 : Y=2**<br/>

반복
◼ 다음 단계의 값을 생성, 누적
◼ Y:수열의 합을 구하는 누적 변수<br/>
**2 : Y=Y+C**<br/>
◼ 동일한 과정으로 반복
    ◼ B를 앞단 A에 복사<br/>
    **3 : A=B**<br/>
    ◼ 새로 생성된 C는 앞단 B에 복사
    **4 : B=C**<br/>




 ~~~
#include <stdio.h> void main(){ 
int A=1, B=1; /* 피보나치 수열의 두 개의 초기값 */ 
int Y=2; /* A, B의 누적 
*/int K, N;
int C; /* 다음 피보나치 수열 */ 
scanf("%d", &N);/* 3항부터 N항까지 피보나치 수열을 생성하고 누적 */ 
for (K=3; K<=N; K++) { 

C=A+B; /* 다음 피보나치 수열 생성 */
Y=Y+C; /* 피보나치 수열의 누적 */
A=B; /* 앞단의 수열에 복사 */
B=C; 
} 

printf("%d\n", Y); /*N항까지 피보나치수열 합 출력*/ }
