## JSX 안에 자바스크립트 값 사용하기 
    




### App.js

~~~javascript
import React from 'react';
import Hello from './Hello';

const element = 'This is JSX'

function App() {
  return (
    <div>
      <Hello />
      {element}
    </div>
  );
}

export default App;
}
~~~

## style 과 className

JSX 에서 태그에 style 과 CSS class 를 설정하는 방법은 HTML 에서 설정하는 방법과 다릅니다.<br/>

우선, 인라인 스타일은 객체 형태로 작성을 해야 하며, background-color 처럼 - 로 구분되어 있는 이름들은<br/> backgroundColor 처럼 camelCase 형태로 네이밍 해주어야 합니다.<br/>

### App.js

~~~javascript
import React from 'react';
import Hello from './Hello';

function App() {
  const element = 'This is JSX';
  const style = {
    backgroundColor: 'black',
    color: 'aqua',
    fontSize: 24, // 기본 단위 px
    padding: '1rem' // 다른 단위 사용 시 문자열로 설정
  }

  return (
    <div>
        <Hello />
      <div style={style}>{element}</div>
    </div>
  );
}

export default App;
~~~

## JSX의 응용

JSX는 표현식이기 때문에 if나 for문 안에서 사용될 수도 있습니다.

~~~javascript
import React from 'react';
import Hello from './Hello';
import './App.css'; //추가


//추가
function hi(name) {
  if(name === 'dd'){
    return "Hello! " + name;
  }
  else {
    return "Who are you?";
  }
}


function App() {
  const element = 'This is JSX';
  const style = {
    backgroundColor: 'black',
    color: 'aqua',
    fontSize: 24, // 기본 단위 px
    padding: '1rem' // 다른 단위 사용 시 문자열로 설정
  }

  return (
    <div>
        <Hello />
        <div style={style}>{element}</div>
        <div className="gray-box"></div>
        {/* 추가 */ }
        <h1> 
          {hi('Choi')}
        </h1>
      
    </div>
  );
}

export default App;

~~~

위 코드처럼 hi함수에 if와 for문을 추가해 보았습니다.
